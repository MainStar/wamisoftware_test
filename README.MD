# Опис проекту
Shape Calculator API — это REST API для расчёта площади и периметра (окружности) геометрических фигур:

# We have only one API endpoint
### /shape/{shapeType}/calculate-are

# Technology stack
Java 8
Spring Boot 2
Maven
Jackson

# Build the application
mvn clean install

# Run the application
mvn spring-boot:run
